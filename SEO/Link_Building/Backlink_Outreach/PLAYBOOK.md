# Backlink Outreach Playbook

> **Tactic:** Backlink Outreach (Resource Page Link Building, Link Insertion / Niche Edits)
> **Difficulty:** Medium | **Scalability:** High | **Link Quality:** Medium–High
> **Time to First Link:** 1–4 weeks per placement

---

## 1. When to Use This Tactic

Use Backlink Outreach when:
- You need to build links to existing content quickly (no new content required)
- You want to earn links from resource pages, curated lists, or "best of" pages in your niche
- You have high-quality content that deserves to be referenced by others
- You want to insert links into already-indexed, already-ranking articles (link insertions / niche edits)
- Client has cornerstone content pages that need authority boosts

**Two Primary Sub-Tactics:**

**1. Resource Page Link Building**
- Target: pages that curate links to helpful resources in a niche ("Best Tools for X," "Ultimate List of Y Resources")
- Ask: "Please add our resource to your list"
- Best for: tools, guides, glossaries, calculators, templates

**2. Link Insertions / Niche Edits**
- Target: existing blog posts ranking for relevant keywords that could naturally reference your content
- Ask: "Can you include a mention/link to our content in this article?"
- Best for: adding depth to already-ranking content, faster results than guest posting

**Do NOT use Backlink Outreach when:**
- The client's content isn't genuinely valuable — editors will reject requests for weak content
- You're sending mass-blast outreach without personalization
- You need links from news/media outlets (use Digital PR)

## 2. Prospecting Criteria & Qualification Framework

### Minimum Acceptance Criteria
| Criteria | Threshold |
|---|---|
| Domain Rating (Ahrefs) | ≥ 30 |
| Monthly Organic Traffic | ≥ 500/mo |
| Spam Score (Moz) | ≤ 15% |
| Topical Relevance | Same or adjacent niche |
| Page Type | Resource page, list article, or relevant blog post |
| Page is indexed and ranking | Confirm via Google: site:domain.com "page title" |
| Site links out to external sources | Yes (check existing article for external links) |
| Not on PBN or link scheme blacklist | Pass |

### Resource Page Qualification Checklist
- [ ] Page title contains "resources," "tools," "links," "best," "top," or similar
- [ ] Page has at least 5 external links already (confirms they link out freely)
- [ ] Links on the page are to real, quality sites (not just sponsors or affiliates)
- [ ] Page has been updated within the last 2 years (not abandoned)
- [ ] Our content would genuinely fit the existing list without being out of place

### Link Insertion Qualification Checklist
- [ ] Article is ranking in top 20 for a keyword relevant to the client's content
- [ ] Article mentions a topic where the client's content adds depth or missing info
- [ ] Current article doesn't link to a competitor's equivalent page already
- [ ] Article has DO-FOLLOW external links to other sites (confirms linking behavior)
- [ ] The site owner/editor is reachable via email

### Prospecting Methods

**Method 1: Google Resource Page Search**
```
[niche keyword] + "resources"
[niche keyword] + "useful links"
[niche keyword] + "best tools"
[niche keyword] + "top [content type]"
intitle:"resources" + [niche keyword]
inurl:resources + [niche keyword]
```

**Method 2: Ahrefs "Best by Links" for Competitor Content**
1. Identify which of the client's pages you want to build links to
2. Search for the target keyword in Ahrefs Content Explorer
3. Find the top-linked articles covering that topic
4. Identify which pages link to those articles — those same pages are your prospects

**Method 3: Competitor Backlink Gap Analysis**
1. Pull backlinks for top 3 competitors (Ahrefs Site Explorer → Backlinks)
2. Filter: "Resource page" type links (look for pages with "resources" in URL or title)
3. These sites have already linked to similar content — high chance of linking to yours too
4. Export and add to your prospect list

**Method 4: "Niche Edit" Prospecting via Ahrefs Content Explorer**
1. Search target keywords in Ahrefs Content Explorer
2. Filter: Published date > 1 year ago (these are established posts open to edits)
3. Filter: DR 30+, Traffic > 500/mo
4. Find articles that cover the same topic as the client's content
5. Identify where a link insertion would naturally fit

## 3. Step-by-Step Outreach Process

### Phase 1: Research & Asset Verification (Days 1–2)

**Step 1: Identify the Client Content Asset**
- Determine exactly which URL on the client's site you want to build links to
- Ensure the page is live, loads fast (< 3 seconds), and is indexed in Google
- Confirm the content is genuinely high-quality: comprehensive, well-formatted, no thin content
- If the content is thin or outdated, update it BEFORE beginning outreach — editors will check it

**Step 2: Build Prospect List**
- Use the prospecting methods above to build a list of 50–150 qualified targets
- For resource pages: Use Google search operators
- For link insertions: Use Ahrefs Content Explorer
- Organize in Airtable or Google Sheets with: URL, domain, DR, traffic, editor email, status

**Step 3: Find Contact Information**
- Check page footer, About page, or Contact page for editor/owner email
- Use Hunter.io or Apollo.io to find verified email for the domain
- For smaller blogs: LinkedIn or Twitter DM is often the fastest path
- Verify emails with NeverBounce or Hunter Verifier before sending batch

**Step 4: Analyze the Target Page**
- For resource pages: note how many links are already on the page, the categories, and the format
- For link insertions: identify the exact sentence or section where your link would naturally fit
- Write a specific insertion suggestion: "In the section on [topic], you could add a link to our [guide/tool] which covers [specific detail they're missing]"

### Phase 2: Outreach (Days 3–10)

**Step 5: Personalize Each Pitch**
- Reference the specific page you're asking about
- Explain WHY your content fits (don't just say "I think this would be a great addition")
- For link insertions: suggest the specific paragraph and context where the link fits
- Keep pitches under 150 words — shorter = higher response rate for outreach

**Step 6: Send Initial Outreach Emails**
- Send in batches of 20–30/day to protect sender domain reputation
- Use warmed outreach email address (not fresh, not client's primary email)
- Personalize the opening line of every email — never copy-paste
- Best send times: Tuesday–Thursday, 8–11 AM recipient's local time

**Step 7: Follow Up**
- Follow-up #1: Day 4–5 after initial send (brief, friendly bump)
- Follow-up #2: Day 9–10 after initial (final attempt, add new value angle)
- Maximum 3 touches total; respect no-responses and move on
- If the contact asks for something in return (reciprocal link, payment), evaluate:
  - Reciprocal link to their site: Only if their site is high-quality and relevant; use judiciously
  - Payment for placement: Avoid — Google's link scheme policies apply; nofollow/sponsored tag required

**Step 8: Handle Replies**
- Positive reply → Confirm the exact URL and anchor text being used; follow up if not published within 2 weeks
- Request for reciprocal link → Evaluate their site quality; accept only if genuinely relevant
- "Not interested" → Thank them, remove from prospect list, do not follow up
- No reply after 3 touches → Mark as exhausted, move to next batch

### Phase 3: Confirmation & Verification (Days 14–30)

**Step 9: Verify Link Placement**
- Once editor confirms the link is live, visit the page yourself
- Check: Is the link do-follow? Contextual? Pointing to correct URL? Using appropriate anchor text?
- Run target URL through Ahrefs to confirm the backlink is indexed
- If the link isn't live within 14 days of confirmation, send a polite follow-up

**Step 10: Track and Monitor**
- Log every live link in your tracking sheet
- Set up Ahrefs Alerts for the client domain to catch any new links automatically
- Monitor monthly for any link removals (some sites periodically clean up external links)
- If a link is removed, attempt one outreach to reinstate it

## 4. Outreach Email Templates

### Template A: Resource Page Addition Request

**Subject:** Addition to your [Niche] resources list?

Hi [First Name],

I came across your resource page on [Topic] — it's one of the more comprehensive lists I've seen in this space.

I wanted to flag a resource that might be a good fit for the list: [Client Content Title] at [URL].

It covers [brief description of what makes it valuable — specific, not generic]. Given that you already include resources like [reference one of their listed resources], I think it would be a useful addition for your readers.

Happy to return the favor if you ever need anything from our end.

[Your Name]
[Title] | [Website]

---

### Template B: Link Insertion Request

**Subject:** Quick suggestion for your [Article Title] post

Hi [First Name],

I've been reading your post on [article topic] — really solid breakdown.

I noticed you mention [specific topic or section in their article]. We recently published a [guide/tool/study] on [specific related topic] that goes deeper on this: [URL]

Would it make sense to add a mention in that section? I think it would add useful context for your readers.

No worries if not — just thought it was a natural fit.

[Your Name]
[Title] | [Website]

---

### Template C: Follow-Up #1

**Subject:** Re: Quick suggestion for your [Article Title] post

Hi [First Name],

Just following up on my note from last week in case it got buried. Happy to share more context on the resource if it helps.

[Your Name]

---

### Template D: Follow-Up #2

**Subject:** Re: Quick suggestion for your [Article Title] post

Hi [First Name],

Last follow-up from me — if the timing isn't right or it's not the right fit, totally understand. I appreciate your time either way.

[Your Name]

---

### Template E: Paying It Forward (Warm Outreach)

**Subject:** Loved your [Article] — quick question

Hi [First Name],

I've been referencing your post on [article topic] a lot recently — even cited it in our latest guide on [client content topic].

I had a quick question: we have a related resource at [URL] that covers [what it covers]. Would you consider including it in your piece? I genuinely think it would complement the section on [specific section].

Either way, keep up the great work — your blog is consistently one of the better resources in [niche].

[Your Name]

## 5. Campaign Tracking & Management

### Tracking Spreadsheet Schema
| Field | Description |
|---|---|
| Prospect URL | Full URL of the target page |
| Domain | Root domain |
| DR | Domain Rating at contact time |
| Traffic | Monthly organic traffic estimate |
| Page Type | Resource page / Link insertion / Other |
| Contact Name | Editor or site owner |
| Contact Email | Verified email |
| Date Pitched | Initial send date |
| Follow-up 1 | Date |
| Follow-up 2 | Date |
| Status | Prospect / Pitched / Replied / Accepted / Live / Rejected / No Response |
| Link URL | Live article URL (once confirmed) |
| Target Page | Client URL being linked to |
| Anchor Text | Anchor used |
| Do-Follow? | Yes / No |
| Date Live | When confirmed live |
| Notes | Reciprocal offer, special conditions, etc. |

### Weekly Workflow
- **Monday:** Prospect 30–50 new targets for the week; verify emails
- **Tuesday:** Send outreach batch #1 (20–30 emails)
- **Wednesday:** Send outreach batch #2 (20–30 emails)
- **Thursday:** Follow up on Day-5 emails from previous week
- **Friday:** Review replies, log status updates, verify any live links

### Scaling the Process
- At 50+ emails/day, use Instantly.ai or Lemlist with rotating sending domains
- Warm up new sending domains for 2–3 weeks before ramping volume
- Use merge tags for personalization at scale: {{first_name}}, {{website}}, {{specific_article}}
- Set up auto-tagging rules in your CRM to track reply categories (Positive / Negative / Question)

## 6. Success Metrics & KPIs

### Primary KPIs
| Metric | Target |
|---|---|
| Response Rate | 10–20% |
| Acceptance Rate (of responses) | 40–60% |
| Overall Conversion (outreach → live link) | 5–12% |
| Average DR of Live Links | ≥ 35 |
| % Do-Follow Links | ≥ 85% |
| Links Built Per Month (per client) | 8–20 |
| Time from First Contact to Live Link | < 21 days |

### Secondary KPIs
- % of links that are contextual (in body, not sidebar/footer)
- % of placed links to client's priority pages (strategic vs random)
- Referring domain growth rate month-over-month
- Client domain rating trajectory (measured monthly)
- Organic traffic change to target pages (measure in GSC, 60-day window)

### Benchmark Response Rates by Industry
- SaaS / Tech: 8–15%
- Health / Wellness: 10–18%
- Finance: 6–12%
- E-commerce / Retail: 5–10%
- Marketing / SEO: 12–20% (most link-savvy editors)

## 7. What to Avoid

### Google Penalty Risks
- **Paying for link insertions (niche edits)** — Untagged paid links violate Google's guidelines. If you pay for a placement, the link must be tagged rel="sponsored" or rel="nofollow"
- **Exact-match anchor text** — Using "best CRM software" as anchor text for a CRM tool is a manipulation signal. Use branded or partial-match anchors
- **Building links to the same page from the same niche cluster** — Over-concentration of links with the same anchor to the same page triggers Penguin-like filters
- **Low-quality or irrelevant resource pages** — A link from "General Internet Resources" with 1,000 unrelated links has zero value and potential negative impact

### Operational Mistakes
- **Sending outreach from a fresh domain** — New domains have no sending reputation; emails go straight to spam. Warm up for 2–3 weeks before ramping.
- **Mass-blasting without personalization** — Response rates collapse below 2% without personalization. Always reference the specific page.
- **Not verifying the link after placement** — Editors sometimes make mistakes (wrong URL, nofollow added). Always verify manually.
- **Accepting reciprocal links from low-quality sites** — A link exchange with a spammy site can hurt the client's profile.
- **Not monitoring for link loss** — Links get removed. Set Ahrefs Alerts and check monthly.
- **Over-concentrating links on one page** — Distribute link building across multiple client pages for a natural profile.
- **Ignoring anchor text diversity** — Aim for: 40% branded, 30% naked URL, 15% partial match, 10% generic ("click here," "this guide"), 5% exact match
