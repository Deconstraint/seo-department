# International SEO — Playbook

## When to Use This
Load this playbook when a client request involves Hreflang, geo targeting, country domains, global content.

## Step-by-Step Process

### Step 1: Audit Current State
- [ ] Assess what exists today
- [ ] Identify gaps and opportunities
- [ ] Prioritize by impact vs effort

### Step 2: Research & Strategy
- [ ] Pull relevant data
- [ ] Define target outcomes
- [ ] Build execution plan

### Step 3: Execute
- [ ] Follow sub-folder playbooks for specific tactics
- [ ] Document all changes made
- [ ] Set tracking in place

### Step 4: Measure & Report
- [ ] Pull metrics after 30 days
- [ ] Compare to baseline
- [ ] Adjust and iterate

## Common Mistakes to Avoid
- Skipping the audit phase
- Executing without baseline metrics
- Not tracking changes made

## Deliverables
- Audit report
- Strategy document
- Execution log
- Monthly performance report
