# Deconstraint SEO Department

**Operational knowledge base for SEO strategy, execution, and client delivery.**

## How to Use

This repo is the operating manual for all SEO work at Deconstraint.

- **Agents:** Read the relevant playbook folder before executing any SEO task
- **Humans:** Browse by section to find process docs, checklists, and tools
- **Clients:** Deliverables are generated from these playbooks

## Trigger Map

| Request | Folders to Load |
|---|---|
| "Audit our site" | Technical_SEO + On_Page_SEO + Analytics/Audits |
| "Content strategy" | Research + Content_SEO + Site_Structure |
| "Why aren't we ranking?" | Research/SERP_Analysis + Competitor_Research |

## Structure

Each folder contains:
- `OVERVIEW.md` — What this area is, why it matters
- `PLAYBOOK.md` — Step-by-step execution process
- `TOOLS.md` — Tools, APIs, scripts for this area
